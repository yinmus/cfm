# CFileManager

легкий, быстрый файловый менеджер (TUI)

**Зависимости:**

`ncurses`, `glibc`

##### Установка
###### Arch/Manjaro (pacman)
```bash
sudo pacman -S ncurses
```
###### Debian/Ubuntu (apt)
```bash
sudo apt install libncurses5-dev libncursesw5-dev
```
###### Fedora (dnf)
```bash
sudo dnf install ncurses-devel
```

___
[**Compile**](Makefile)

- make help

**Usage:**

- Alt + A help
